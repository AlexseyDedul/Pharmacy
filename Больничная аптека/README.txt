На прилагаемом электронном носителе информации расположены следующие директории и файлы: 
1)	Директория «DOC» – файл «П31_Дедюль_А.Н._ИC_Больничная аптека.doc» представляет собой текстовый документ с пояснительной запиской. 
2)	Директория «PRG» содержит исходный код проекта Аpteka.exe. 
3)	Директория «EXE» содержит финальную версию исполняемых файлов проекта Аpteka.exe.
4)	Файл «Readme.txt» содержит описание содержания электронного носителя.
